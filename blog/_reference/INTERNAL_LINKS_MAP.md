## 교차 링크 패턴

### 같은 메커니즘 교차 (독자가 "이 회사도 같은 구조네?" 하는 순간)

| 출발 글 | → 도착 글 | 앵커 텍스트 | 이유 |
|---------|----------|-----------|------|
| SK하이닉스 (고정비 레버리지) | → 삼양식품 | "삼양식품의 불닭도 같은 고정비 레버리지" | 같은 메커니즘, 다른 산업 |
| 한화에어로 (고정비 레버리지) | → 삼양식품 | "삼양식품 고정비 레버리지와 같은 구조" | 본문에 이미 언급 |
| 한화에어로 (수주잔고→매출) | → HD현대일렉트릭 | "HD현대일렉트릭도 수주잔고 메커니즘" | 수주 구조 비교 |
| HD현대일렉트릭 (적자→흑자) | → 두산에너빌리티 | "두산에너빌리티도 턴어라운드" | 턴어라운드 비교 |
| 고려아연 (자사주=무기) | → 크래프톤 | "크래프톤도 자사주 소각 전략" | 자사주 활용 비교 |
| 에이피알 (DTC 마진) | → 셀트리온 | "셀트리온도 직접 판매(짐펜트라)" | 직접 판매 전략 비교 |
| 크래프톤 (현금 추적) | → 고려아연 | "고려아연도 현금이 어디로 갔는가" | 자본배분 추적 비교 |

### 같은 시리즈 인접 (자동 — 배너에서 처리)

시리즈 링크는 각 글 상단 배너에서 이미 11편 전체를 연결. 추가 불필요.

## 고립 감지

글을 추가/개선할 때마다 `scripts/blog/audit_seo.py`로 inbound 링크 수를 체크. inbound 0인 글은 다른 글에서 링크 삽입.

## 규칙

- 글 작성 시 **최소 2개** 교차 링크 (시리즈 링크 제외)
- 같은 메커니즘 교차가 가장 자연스러움
- 강제로 넣지 않음 — 본문 흐름에서 자연스러운 곳에만
