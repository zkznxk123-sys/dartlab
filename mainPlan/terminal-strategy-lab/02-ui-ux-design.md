# 02. UI/UX Design — 확정 설계

상태: v0.1 (2026-06-16). 거처 = `ui/packages/surfaces/src/terminal/`.

> 원칙: 차트가 1차 표면(마커+에쿼티). 컨트롤 = **차트-스코프 인셋**(시뮬 05 §4 패턴), 별도 패널/사라지는 팝오버 폐기. 상세 = 포커스 1전략 다이얼로그. 가독 하한 10.5px.

---

## 1. 공간 모델 — 3개 표면, 하나의 차트

| 표면 | 역할 | 거처 |
|---|---|---|
| **차트 메인 페인** | 캔들 + 포커스 전략 진입/청산 마커 + 리밸런싱 ⚖ 마커 + (P5)미래 fan | candle_pane (btLayer BT_TRADES draw) |
| **차트 에쿼티 서브페인** | N전략 + 조합 + B&H 곡선(**공유 절대축**) + 조합 MDD 음영 | pane_BT (BT_EQUITY 공유축 draw) |
| **전략 콘솔 인셋** | N전략 추가/설정/가중치/스탑 + 리플레이 컨트롤 | 차트 좌상 오버레이(접힘 가능) |
| **헤드라인 strip** | 조합 vs B&H + 포커스 전략 + 정직 라벨 + [상세] | 차트 직하 1줄 |
| **상세 다이얼로그** | 포커스 1전략 심층(거래·MAE/MFE·낙폭·가정) | scrModal (on-demand) |

진입 = ChartMenus `[전략]` 토글 → 콘솔 인셋 펼침 + 차트에 마커/에쿼티 표시. 설정·결과·차트가 *한 시야*.

## 2. P1 — 다전략 캔버스 (목표 종착의 골격)

```
┌─ CenterStack ─────────────────────────────────────────────────────────┐
│ ChartMenus  …  [전략 ▣](활성 amber)                          370,000   │
│ ┌─전략 콘솔──────────┐                                                  │
│ │ ◆ A 골든크로스   ⚙✕│        ▲A          ▼A     ╱╲          330,000   │
│ │   fast 20 slow 60  │   ╱╲     ▲B    ╱╲   ╱  ╲  ▼B                    │ ← 포커스=A 마커
│ │ ◆ B RSI 반전     ⚙✕│  ╱  ╲___╱    ╲_╱    ╲___╱  ╲__                  │
│ │   period 14        │ ╱                                               │
│ │ ＋ 전략 추가 (2/3)  │                                    270,000      │
│ │ ─────────────────  │                                                 │
│ │ 조합: 동일가중      │                                                 │
│ │ 비용 ✓ · 검증 70:30 │                                                 │
│ │ ▶ ❚❚  1× 2.5×  ⏮   │                                                 │
│ └────────────────────┘                                                 │
│ ┌─ 에쿼티 (공유 절대축, 시작 100) ─────────────────────────────────┐  │
│ │ 조합 ━━━━━━━━━━━━━━━━━━ +22%                                     │  │ ← 굵게(주연)
│ │ A   ──────────────────── +30% (녹)                              │  │
│ │ B   ···················· +12% (보라)                            │  │
│ │ B&H - - - - - - - - - -  +12% (회색 점선, 동일비용)             │  │
│ │ ░░░░ 조합 MDD 음영 -14%  (A 단독 MDD -23%보다 얕음 = 분산효과)   │  │ ← 시각 증거
│ └────────────────────────────────────────────────────────────────┘  │
│ A +30% · 조합 +22% vs 보유 +12% │ ⚠ 같은 종목 = 타이밍 분산(자산 분산 아님) · 추천 아님 │
│                                                            [전략 상세 ▸] │
└────────────────────────────────────────────────────────────────────────┘
```

핵심 UX 결정:
- **포커스 전략**(콘솔에서 ◆ 클릭) = 캔들 마커가 그 전략만(N≥2 클러터 방지). 에쿼티는 항상 전 곡선.
- **조합 곡선이 주연**(2px), 개별은 1.2px, B&H 회색 점선. **승자 강조·정렬 금지**(중립 렌더).
- **분산효과 시각 증거**: 조합 MDD 음영이 개별보다 얕으면 그 자체가 "조합의 값"(헤드라인에 "조합 MDD -14% < A -23%" 미니 라벨). 이게 "고정가중=시시함" 평가절하를 막는 핵심.
- **selection 경고**: 콘솔 하단 "N전략을 같은 데이터로 비교 = 사후선택 편향. 위에 있는 곡선이 미래 최고 아님."

## 3. P2 — 포지션·거래 정밀 (상세 다이얼로그 = 포커스 1전략)

콘솔 ⚙ → 전략별 스탑 설정(손절%·익절%·트레일%·N봉 청산). 캔들 마커: 손절 청산 = ✖(적), 신호 청산 = ▼.

[전략 상세 ▸] → **포커스 1전략 다이얼로그**(N-aware 금지, 단수 유지):
```
┌─ 전략 상세 — A 골든크로스 ──────────────[탭: A│B│조합]──────────[✕]─┐
│ A +30.2% vs 보유 +12.1% · 초과 +18.1%p                              │
│ ┌─ 자산 곡선 + 진입/청산/손절 ──────────────────────────────────┐  │
│ │ (에쿼티 + MAE/MFE 음영)                                        │  │
│ └────────────────────────────────────────────────────────────────┘  │
│ ┌개요──────┐┌거래 분석──────────────┐┌낙폭/회복──────┐┌가정──────┐  │
│ │CAGR Sharpe││기대값 +1.2% · R 1.8   ││MDD -23% 87일  ││t+1시가   │  │
│ │Sortino    ││MAE/MFE 산점도         ││Calmar 1.3     ││비용 ON   │  │
│ │Calmar MAR ││연속승패 · 보유분포     ││Ulcer 8.2      ││배당미반영 │  │
│ │베타 알파IR││수익 히스토그램·CSV     ││underwater곡선  ││무수정주가 │  │
│ └──────────┘└──────────────────────┘└───────────────┘└──────────┘  │
│ 거래표(진입·진입가·청산·MAE·MFE·R·보유일·사유) 행클릭→차트 봉       │
│ ⚠ 단일구간 in-sample · OOS 검증 70:30 동반 · 추천 아님              │
└──────────────────────────────────────────────────────────────────────┘
```
- 조합 탭 = equity 곡선 + equity 메트릭만(**거래표·CSV 없음** — combo 거래 0, N.A. 명시).
- 거래 분석 = expectancy·R-multiple·MAE/MFE·streak·분포(canon 추가, 현 제품에 없음).

## 4. P3 — 커서바인딩 리밸런싱

콘솔에 가중치 바(전략별 %, 합 100% 잠금). **리플레이 일시정지 상태에서만 활성**(재생 중·전체차트 모드엔 잠금 + "되감아 그 시점에서만" 라벨).
```
┌─전략 콘솔──────────────┐
│ ◆ A  [████████░░] 80%  │ ← 일시정지 시만 드래그
│ ◆ B  [██░░░░░░░░] 20%  │
│ ▶ ❚❚ │ t=2025.09.12    │ ← 현재 커서 시점(t까지만 보임)
│ ⚖ 리밸런싱 3회         │ ← append-only(되감아 수정=재포크)
└────────────────────────┘
 에쿼티: 조합 곡선에 ⚖ 세로 점선(리밸 시점) + 구간별 유효가중 [80/20][50/50]
 ⚠ 같은 구간 N회 재생 = in-sample 과적합 (반복재생 카운터 노출)
```
정직 4조건(04 §): 커서 바인딩 · append-only · 반복재생 카운터 · 자유 슬라이더는 사전규칙 보완(P3는 *재량 walk*라 슬라이더 허용하되 커서 잠금).

## 5. P5 — 미래 연속

리플레이 끝봉 도달 → [미래로 ▶▶] → `mode:simulate`. asOf 우측 fan band, 조합 곡선이 asOf에서 끊김 없이 fan으로.
```
 에쿼티 ──── asOf ────────────
 조합 ━━━━━╱⚖━━━━●═════╗ ░P75░     ● 바통점(포트 상태→시뮬 초기조건)
 A   ──────────● ╔═══╝▒P50▒          ═ 점선=가정 fan
 B   ··········● ╚═══╗ ░P25░
 [live = 실현]  │ [simulate = 가정 경로·투자권유 아님·conf 라벨]
              asOf
```
미래 = 점선·반투명·"가정 구간" 워터마크·단일 path 금지(밴드/다분기). path 데이터 = 시뮬 코어 졸업 후.

## 6. 타이포·색·인터랙션 규칙

| 역할 | 크기 |
|---|---|
| L1 헤드라인 값 | 16–17px bold mono |
| L2 KPI 카드값 | 16–17px · 칩/표 본문 11.5–12px |
| L3 라벨·섹션헤더·정직 각주 | 10.5px(하한) |
| 금지 | ≤10px(값·라벨). SVG 축 눈금만 예외 |

- 색: `STRAT_COLORS`(≤3, 녹/보라/cyan) + 조합 amber 굵게 + B&H 회색 점선 + 손절 적색 ✖. 손실=`--dn`. **승자 강조 금지**(굵기·정렬·색에 우열 신호 0). 전략 열위 시 dim(초록 축포 금지).
- 인터랙션: 콘솔 ◆=포커스 전환 / ⚙=스탑 설정 / ✕=전략 삭제 / ＋=추가(≤3). 거래행 클릭→차트 봉 센터링(리플레이 유지). 리밸런싱 슬라이더=일시정지 시만. 인셋 접힘=헤드라인만(현 strip 역할).
- 정직 표면 상존(닫기불가): selection 경고 · 단일종목 분산 라벨 · 표본<10 빨강 · 배당미반영+무수정주가 · "과거 가정 노출형 시뮬레이션·추천 아님" 푸터 · "전문가급" 문구 0.

## 7. 컴포넌트 매핑 (재사용/신규)
- **재사용**: `scrModal/scrHead/scrClose`(다이얼로그), `.bdGrid/.bdCard`(KPI), `.bdTable`(거래표), 색토큰, `compareOverlay` draw 패턴(참고), 리플레이 컨트롤(05 공유).
- **신규**: `StrategyConsole.svelte`(인셋), `btViz/`(EquityCurve 공유축·MaeMfeScatter·ReturnHistogram·MonthlyHeatmap·SensitivitySweep), `BtConfig`→콘솔 N행 흡수, `BacktestStrip`→헤드라인 흡수, `BacktestDialog`→포커스1 탭.
