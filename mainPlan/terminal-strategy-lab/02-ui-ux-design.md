## 1.5 ★차트 시각화 개념 — 정렬된 시간 레인 스택 (핵심)

> 한 줄: **조작 패널에서 만든 추상 규칙이 차트 위에서 구체적 시간 패턴이 된다.** "RSI<30 AND 가격>MA200 AND Piotroski≥6"를 입력하면 — 3개 조건이 차트에서 각자 켜지고, 어디서 AND로 합쳐져 진입이 터지는지가 *눈에 보인다*. 사용자는 "왜 여기선 안 샀지?"를 즉시 안다(어느 조건이 꺼져 있었나).

주가차트가 전략의 *모든 구성요소*를 같은 x축(시간)에 **정렬된 레인**으로 쌓는다:

```
┌─ ① 가격 페인 (candle_pane) ───────────────────────────────────────┐
│ ░░░░░ 펀더 게이트 배경 음영 (Piotroski≥6 구간=초록 틴트, 분기 계단) ░░░│ ← ★moat 시각화
│   캔들  ▲진입  ✖손절  ▼신호청산  (포커스 전략)                       │
│   + 조건에 쓰인 지표만 오버레이 (MA200·BB…) — 안 쓴 지표는 안 그림    │
├─ ② 조건 레인 (조건 빌더 결과 — 작은 on/off 스트립, 빌더와 1:1) ──────┤ ← ★규칙 가시화
│   RSI<30       ▮▮░░░░▮▮░░░▮▮░░░░                                    │
│   가격>MA200    ░░░▮▮▮▮▮▮▮▮▮▮▮░░░                                   │
│   Piotroski≥6   ░░░░░▮▮▮▮▮▮▮▮▮▮▮  (분기 계단·PIT 근사)               │
│   = 진입(AND)   ░░░░░░░▮▮░░░▮░░░░  ──→ ①의 ▲진입과 수직 정렬          │
├─ ③ 에쿼티 페인 (공유 절대축, 시작 100) ────────────────────────────┤
│   조합 ━━  A ──  B ···  B&H - -   + 조합 MDD 음영                   │
├─ ④ 낙폭 레인 (언더워터) ──────────────────────────────────────────┤
│   ▔▔╲▁▁╱▔▔                                                         │
└────────────────────────────────────────────────────────────────────┘
        리플레이 커서 t ──┃ (전 레인 동시 절단·동기 재생 — look-ahead 상속)
```

**레인별 시각화 규칙**:
1. **가격 페인** — 캔들 + 포커스 전략 마커(진입▲/손절✖/신호청산▼ 색 구분) + *조건에 실제 쓰인* 지표 오버레이만(클러터 차단). ★**펀더 게이트 = 배경 음영**: "재무가 튼튼했던 구간"(예 Piotroski≥6)이 가격 뒤에 초록 틴트로 깔린다 — 분기 계단, **PIT 근사(공시일 `rcept_dt` 이후만 켜짐)**. 가격 백테스터엔 없는 panel moat 레이어.
2. **조건 레인** — 조건 빌더의 각 조건이 시간축 on/off 스트립. 마지막 "= 진입(AND/OR)" 합성 레인이 ①의 진입 마커와 *수직 정렬*. 조건이 곧 그림 → "이 규칙이 언제 충족됐나"가 직관.
3. **에쿼티 페인** — 다전략+조합+B&H 공유 절대축(§1.1 함정 회피).
4. **낙폭 레인** — combo 언더워터.

**상호작용**: 콘솔에서 조건 추가/수정 → 해당 조건 레인 즉시 추가/갱신 → AND 레인·진입 마커·에쿼티 동시 재계산. 리플레이 커서가 전진하면 전 레인이 같은 t까지 동기 절단(전략이 "그 시점까지만 보고" 결정하는 정직 walk가 *모든 레인에서 동시에* 보인다). 레인 = klinecharts 서브페인(`figures:[]` + custom draw)로 박음(btLayer N슬롯 패턴 확장). 조건 레인 LOD: 줌아웃 시 합성(AND) 레인만, 줌인 시 개별 조건 펼침.

**왜 이게 "전문가급"인가**: 블랙박스 "전략 실행 → 수익률 숫자"가 아니라, *규칙의 각 부품이 시간 위에서 어떻게 작동하고 어디서 결합되는지*를 분해해 보여준다. 이게 조작 패널(추상 규칙 조립)과 차트(구체 시간 패턴)를 한 몸으로 묶는 개념 — 6 드롭다운이 절대 줄 수 없는 것.

```
│ └────────────────────────────────────────────────────────────────┘  │
│ A +30% · 조합 +22% vs 보유 +12% │ ⚠ 같은 종목 = 타이밍 분산(자산 분산 아님) · 추천 아님 │
│                                                            [전략 상세 ▸] │
└────────────────────────────────────────────────────────────────────────┘
```

핵심 UX 결정:
- **포커스 전략**(콘솔에서 ◆ 클릭) = 캔들 마커가 그 전략만(N≥2 클러터 방지). 에쿼티는 항상 전 곡선.
- **조합 곡선이 주연**(2px), 개별은 1.2px, B&H 회색 점선. **승자 강조·정렬 금지**(중립 렌더).
- **분산효과 시각 증거**: 조합 MDD 음영이 개별보다 얕으면 그 자체가 "조합의 값"(헤드라인에 "조합 MDD -14% < A -23%" 미니 라벨). 이게 "고정가중=시시함" 평가절하를 막는 핵심.
- **selection 경고**: 콘솔 하단 "N전략을 같은 데이터로 비교 = 사후선택 편향. 위에 있는 곡선이 미래 최고 아님."

## 4. P3 — 커서바인딩 리밸런싱

콘솔에 가중치 바(전략별 %, 합 100% 잠금). **리플레이 일시정지 상태에서만 활성**(재생 중·전체차트 모드엔 잠금 + "되감아 그 시점에서만" 라벨).
```
┌─전략 콘솔──────────────┐
│ ◆ A  [████████░░] 80%  │ ← 일시정지 시만 드래그
│ ◆ B  [██░░░░░░░░] 20%  │
│ ▶ ❚❚ │ t=2025.09.12    │ ← 현재 커서 시점(t까지만 보임)
│ ⚖ 리밸런싱 3회         │ ← append-only(되감아 수정=재포크)
└────────────────────────┘
 에쿼티: 조합 곡선에 ⚖ 세로 점선(리밸 시점) + 구간별 유효가중 [80/20][50/50]
 ⚠ 같은 구간 N회 재생 = in-sample 과적합 (반복재생 카운터 노출)
```
정직 4조건(04 §): 커서 바인딩 · append-only · 반복재생 카운터 · 자유 슬라이더는 사전규칙 보완(P3는 *재량 walk*라 슬라이더 허용하되 커서 잠금).

## 5. P5 — 미래 연속

리플레이 끝봉 도달 → [미래로 ▶▶] → `mode:simulate`. asOf 우측 fan band, 조합 곡선이 asOf에서 끊김 없이 fan으로.
```
 에쿼티 ──── asOf ────────────
 조합 ━━━━━╱⚖━━━━●═════╗ ░P75░     ● 바통점(포트 상태→시뮬 초기조건)
 A   ──────────● ╔═══╝▒P50▒          ═ 점선=가정 fan
 B   ··········● ╚═══╗ ░P25░
 [live = 실현]  │ [simulate = 가정 경로·투자권유 아님·conf 라벨]
              asOf
```
미래 = 점선·반투명·"가정 구간" 워터마크·단일 path 금지(밴드/다분기). path 데이터 = 시뮬 코어 졸업 후.

## 6. 타이포·색·인터랙션 규칙

| 역할 | 크기 |
|---|---|
| L1 헤드라인 값 | 16–17px bold mono |
| L2 KPI 카드값 | 16–17px · 칩/표 본문 11.5–12px |
| L3 라벨·섹션헤더·정직 각주 | 10.5px(하한) |
| 금지 | ≤10px(값·라벨). SVG 축 눈금만 예외 |

- 색: `STRAT_COLORS`(≤3, 녹/보라/cyan) + 조합 amber 굵게 + B&H 회색 점선 + 손절 적색 ✖. 손실=`--dn`. **승자 강조 금지**(굵기·정렬·색에 우열 신호 0). 전략 열위 시 dim(초록 축포 금지).
- 인터랙션: 콘솔 ◆=포커스 전환 / ⚙=스탑 설정 / ✕=전략 삭제 / ＋=추가(≤3). 거래행 클릭→차트 봉 센터링(리플레이 유지). 리밸런싱 슬라이더=일시정지 시만. 인셋 접힘=헤드라인만(현 strip 역할).
- 정직 표면 상존(닫기불가): selection 경고 · 단일종목 분산 라벨 · 표본<10 빨강 · 배당미반영+무수정주가 · "과거 가정 노출형 시뮬레이션·추천 아님" 푸터 · "전문가급" 문구 0.

## 7. 컴포넌트 매핑 (재사용/신규)
