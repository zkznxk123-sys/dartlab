# finance/rawMaterial — 원재료 및 생산설비

> **상태: 실무 투입** (2026-03-07)

## 기능
사업보고서 "3. 원재료 및 생산설비" 섹션에서 3가지 데이터 추출.

### 1. 원재료 매입 현황
### 2. 유형자산 변동 (생산설비)
### 3. 시설투자 현황

## API
```python
from dartlab.finance.rawMaterial import rawMaterial

result = rawMaterial("005930")  # -> RawMaterialResult | None
```

## 파싱 전략

### 원재료
1. 헤더 직접 감지: (매입액|투입액) + (품목|원재료|부문) 조합
2. `_findHeaderIndices()`로 컬럼 인덱스 동적 매핑
3. shifted 행 감지: segment 생략 시 왼쪽으로 밀린 데이터 역산
4. 합쳐진 "매입액(비율)" 셀: `1,483,067 (43.8%)` 패턴 분리
5. 연도별 헤더 (`제N기`, `20XX년`) 패턴 지원
6. 분할 헤더: row1(품목/구분) + row2(매입액/비율) 2행 헤더 병합
7. 비율>100 shifted 감지: 비율칸 값이 100 초과면 shifted로 판단
8. 연도 컬럼 자동 ratio 제외: 매입액(비율) 합쳐진 헤더, 연도 헤더 제외
9. 생산 테이블 필터: 생산능력/수량 키워드 있는 헤더 제외
10. 숫자 item 필터: item이 순수 숫자면 shift된 데이터로 판단, 제외
11. 각주 참조 필터: (주N) 패턴은 parseAmount에서 None 반환
12. amount 없는 헤더 제외: amount 컬럼 미발견 시 테이블 진입 차단

### 생산설비
1. 자산명 3개 이상 + "계/합계" 포함 행을 헤더로 감지
2. `COL_KEYS`로 한글 자산명 → 영문 키 매핑
3. "기말장부금액"/"기말 순장부금액"/"기말" 행에서 기말잔액 추출
4. 감가상각비/CAPEX도 동일 테이블에서 추출

### 시설투자
1. "시설투자"/"투자계획" 섹션 감지
2. "구분" + "투자" 헤더 탐지
3. 각 행의 마지막 숫자를 투자금액으로 추출
