# finance/bond 현황

## 개요
사업보고서 "증권의 발행을 통한 자금조달에 관한 사항" 섹션에서 채무증권 발행실적 추출.

## 파일 구성
| 파일 | 역할 |
|------|------|
| types.py | BondIssuance, BondResult 데이터클래스 |
| parser.py | 10셀 채무증권 테이블 파서 (parseBondTable) |
| pipeline.py | bond(stockCode) 파이프라인 |
| __init__.py | 공개 API export |

## 성능
- 전체 테스트: 100% (102개 파싱 성공 + 125개 해당없음, 실패 0)

## 데이터 구조
10셀 표준 구조:
[0]발행회사 [1]증권종류 [2]발행방법 [3]발행일자 [4]권면총액
[5]이자율 [6]평가등급 [7]만기일 [8]상환여부 [9]주관회사

## 알려진 제한
- 단위(백만원/천원/원)는 종목마다 다름
- 금호전기 등 일부 종목은 원 단위로 기재 (금액이 매우 큼)
- 첫 번째 채무증권 발행실적 테이블만 파싱 (중복 테이블은 미처리)
