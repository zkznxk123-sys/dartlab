# finance/subsidiary 현황

## 개요
사업보고서 "타법인출자 현황(상세)" 섹션에서 자회사/투자 포트폴리오 데이터 추출.

## 파일 구성
| 파일 | 역할 |
|------|------|
| types.py | SubsidiaryInvestment, SubsidiaryResult 데이터클래스 |
| parser.py | 16셀 테이블 파서 (parseSubsidiaryTable) |
| pipeline.py | subsidiary(stockCode) 파이프라인 |
| __init__.py | 공개 API export |

## 성능
- 전체 테스트: 100% (181개 파싱 성공 + 47개 해당없음, 실패 0)
- 10개 대표 종목: 10/10 검증 완료

## 데이터 구조
16셀 표준 구조:
- [0] 법인명 [1] 상장여부 [2] 최초취득일 [3] 출자목적 [4] 최초취득금액
- [5-7] 기초잔액 (수량/지분율/장부가)
- [8-10] 증감 (취득수량/금액/평가손익)
- [11-13] 기말잔액 (수량/지분율/장부가)
- [14-15] 재무현황 (총자산/당기순손익)

## 알려진 제한
- 단위(백만원/천원/원)는 종목마다 다름 — 기준일 행의 단위 정보는 아직 추출하지 않음
- 합계 행은 파싱하지 않음 (개별 항목 합산으로 계산 가능)
